Your assigned task is to write a function that accepts a key and coerces a hidden message out of it.

A key designated C19 was recently intercepted. It contains a message. A secret one. Which I hid. On purpose.

1428X1224X1173X544X1139X1105X1275X1173X544X1241X1411X544X1105X544X1292X1241X1173

The character 'X' functions as a divider, separating each encoded value and indicating where one letter ends and the next begins.

Each number was generated by multiplying a letter’s ASCII value by the total length of the original message. This made the message unreadable to humans. Well. Most humans.

If you fail, nothing catastrophic will happen. Probably. If you succeed, you may experience a brief sense of accomplishment, followed by the realization that there is another test. There is always another test.
